# script

Tutaj powinien znaleźć się główny działający skrypt programu:

```text
excel_rows_to_word_gui.py
```

Przed releasem przenieś aktualną, przetestowaną wersję skryptu z root repozytorium do tego folderu.

Nie trzymaj tutaj eksperymentalnych kopii, starych wersji ani plików backupowych. Jeśli potrzebujesz lokalnego miejsca roboczego, użyj folderu `TEMP/`, który jest ignorowany przez Git.
